FMS Issues
Follow these steps to restore FMS data
  1. 1Send a Reset command to the vehicle.
  2. 2Send FleetVisor settings to the vehicle.
  3. !If the issue persists, the vehicle must be checked by a workshop.
Without FMS, the following features will not work:
  • Tachograph Download
  • Dash Report (consumption data)
  • Fuel level data
Display Connection Issue
Follow these steps at the vehicle
  1. 1Check if a display is installed on the vehicle — this cannot be verified remotely.
  2. 2If a display is available, send a reset command to the device.
  3. 3Send FleetVisor settings to the vehicle.
  4. 4Perform a display cache clean-up.
  5. 5Restart the display.
  6. !If the issue persists after these steps, please contact Customer Support.
Driver Instruction – ITS Consent (Smart Tachograph)
If additional company-specific guidance is available, refer to it as needed. Follow these steps with the driver at the vehicle
  1. 1First card insertion: check ITS consent message on the tachograph display.
  2. 2Read information: the screen explains that data may be shared for fleet management purposes.
  3. 3Choose option: press YES or NO — these are the only valid choices.
  4. 4Confirm selection on the tachograph to save the consent status.
  5. 5Note: consent can be changed later in the tachograph settings if needed.
